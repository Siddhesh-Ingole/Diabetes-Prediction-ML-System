# Diabetes Prediction ML System

## 📌 Description
This project is a Machine Learning-based system developed as part of my AI & ML learning journey. It predicts whether a person is diabetic or non-diabetic based on medical inputs and provides probability insights. The system focuses on applying ML concepts to a real-world healthcare problem.

## 🚀 Technologies Used
- Python  
- Pandas (Data Handling)  
- Scikit-learn (Machine Learning)  
- Logistic Regression (Classification Model)  
- StandardScaler (Data Preprocessing)  
- Joblib (Model Saving)  

## ⚙️ Functionality
- Takes user medical inputs  
- Predicts diabetes risk using ML model  
- Displays prediction with probability  
- Applies feature scaling for better performance  
- Stores prediction history  
- Exports history to CSV file  
- Provides basic health suggestions  

## 📊 Model Details
- Algorithm: Logistic Regression  
- Type: Binary Classification  
- Evaluation: Accuracy & Confusion Matrix  

## 🎯 Learning Outcome
- Understood complete ML workflow (data → preprocessing → training → prediction)  
- Learned feature scaling and model evaluation techniques  
- Built a real-time prediction system with user interaction
